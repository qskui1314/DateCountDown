# DateCountDown
日期倒计时，可自行设置时间、背景。  
显示从当天到截止日期的剩余工作日，截止日期在js中配置。

---
Update 2020.10.10
拆分为横版和竖版  
横版是1920*1080  
竖版是1080*1920  
autoboot.bat放到开机自启的文件夹里，可以实现开机自动调用chrome打开网页，并自动全屏显示
